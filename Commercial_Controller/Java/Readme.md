## Getting started

### Prerequisites

- .Net ^v8

### Installation

1. Clone the repo
  ```sh
  git clone https://github.com/Spyro119/Rocket_Elevators_Controllers/
  ```
2. Cd into the Java directory
  ```sh
  cd Rocket_Elevators_Controllers/Commercial_Controller/Java/
  ```
3. Run the app 
  ```sh
  javac *.java
  ```
